# 自动填入插件和插件选项.py

1. 将脚本与gui-config.json文件放到Shadowsocks.exe的目录中
2. 运行Shadowsocks.exe并启动系统代理后退出Shadowsocks.exe
3. 最后运行脚本即可自动填写插件以及插件选项

本脚本默认插件选项为:
```
obfs=tls;obfs-host=www.mircosoft.com
```

你可以自行修改py脚本
